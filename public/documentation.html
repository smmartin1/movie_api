Returns URL HTTP Method Query Parameters Requests Response
List of movies /movies GET - None JSON containing data of all the movies.
List of a specific movie /movies/:title GET :title None JSON object containing data of a specific movie. Example: 
								
{
	title: '...',
	description: '...',
	genre: {
		name: '...',
		description: '...'
	},
	director: {
		name: '...',
		bio: '...',
		birth: '...',
		death: '...'
	},
	imageURL: '...'
}
List of movie genres /movies/genre/:genre GET :genreName None JSON containing data of a specific movie genre. Example: 
								
{
	genre: {
		name: '...',
		description: '...'
	}
}
Director Biography /movies/director/:director GET :directorName None JSON containing data of a director. Example: 
								
{
	director: {
		name: '...',
		bio: '...',
		birth: '...',
		death: '...'
}
									}
Adding a new user /users POST - None JSON object containing data of the new user. Example: 
								
{
	id: '...',
	name: '...',
	favoriteMovies: []
}
Updating a current user by their id /users/:id PUT :id None JSON object containing the updated info of the current user. Example: 
								
{
	id: '...',
	name: '...',
	favoriteMovies: []
}
Adding a movie to a current user's list /users/:id/:movieTitle POST :movieTitle None [movieTitle] has been added to user [id]'s arrary
Removing a movie to a current user's list /users/:id/:movieTitle DELETE :movieTitle None [movieTitle] has been removed from user [id]'s arrary
Removing a current user /users/:id DELETE :id None User [id] has been deleted.
