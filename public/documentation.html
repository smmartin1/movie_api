Returns URL HTTP Method Query Parameters Requests Response
List of movies /movies GET - None JSON containing data of all the movies.
Movie Information based on Title /movies/:Title GET :title None JSON object containing data of a specific movie. Example: 
								
{
	title: '...',
	description: '...',
	genre: {
		name: '...',
		description: '...'
	},
	director: {
		name: '...',
		bio: '...',
		birth: '...',
		death: '...'
	},
	imageURL: '...'
}
List of movie genres /genre/:Name GET :name None JSON containing data of a specific movie genre. Example: 
								
{
	genre: {
		name: '...',
		description: '...'
	}
}
Director Biography /director/:Name GET :Name None JSON containing data of a director. Example: 
								
{
	director: {
		name: '...',
		bio: '...',
		birth: '...',
		death: '...'
}
									}
Adding a new user /users POST - None JSON object containing data of the new user. Example: 
								
{
	Username: '...',
  Password: '...',
  Email: '...',
  Birthday: MMDDYYYY,
  FavoriteMovies: [...]
}
Updating a current user by their id /users/:Username PUT :Username None JSON object containing the updated info of the current user. Example: 
								
{
	Username: '...',
  Password: '...',
  Email: '...',
  Birthday: MMDDYYYY,
  FavoriteMovies: [...]
}
Adding a movie to a current user's list /users/:Username/movies/:MovieID POST :MovieID None Favorite Movies: ['...', '...']
Removing a movie to a current user's list /users/:Username/movies/:MovieID DELETE :MovieID None Favorite Movies: ['...']
Removing a current user /users/:Username DELETE :Username None Username was deleted.
